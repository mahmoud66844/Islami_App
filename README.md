# Islami App 🕌

## Introduction

**Islami App** is a mobile application developed using **Flutter** that aims to provide essential Islamic resources in a simple and user-friendly interface. The application is designed to help Muslims access important Islamic content such as the Holy Quran, Hadith, supplications, and other spiritual tools that assist in daily worship.

The project demonstrates how Flutter can be used to build modern mobile applications that combine useful content with an intuitive design. It focuses on creating a clean and organized structure while delivering a smooth user experience for reading and navigating Islamic materials.

Islami App serves as a practical example of a religious mobile application that can support users in strengthening their connection with their faith through digital tools.

---

# Project Overview

The Islami App provides a digital platform that allows users to access different Islamic resources from a single application. Instead of using multiple apps for Quran reading, Hadith browsing, or daily supplications, users can find these features integrated into one simple application.

The main goal of this project is to build a lightweight and accessible Islamic application that can be used daily by Muslims for learning, reading, and remembering Islamic teachings.

The application focuses on usability and simplicity so that users of different ages can easily navigate and benefit from the available content.

---

# Main Features

## Holy Quran Section

The application allows users to read the **Holy Quran** directly from the mobile app. The Quran section enables users to browse different surahs and read verses in a clear and readable format.

This feature makes it easier for users to access Quranic content anytime and anywhere.

---

## Hadith Collection

The app also provides a section dedicated to **Hadith**, where users can explore authentic sayings of the Prophet Muhammad (peace be upon him). These hadiths help users learn more about Islamic teachings and apply them in their daily lives.

The hadiths are organized in a way that makes them easy to read and navigate.

---

## Tasbeeh (Digital Tasbih Counter)

A **digital tasbeeh counter** is included to help users track their dhikr (remembrance of Allah). Users can increment the counter while reciting phrases such as:

* SubhanAllah
* Alhamdulillah
* Allahu Akbar

This feature allows users to keep track of their supplications conveniently.

---

## Islamic Radio / Audio Content

The application may also include an **Islamic audio or radio section**, allowing users to listen to Quran recitations or Islamic lectures while using the app.

This provides an additional way for users to engage with Islamic content.

---

## Clean and Simple Interface

The UI of the application is designed to be simple and easy to navigate. The layout ensures that users can quickly move between different sections such as Quran, Hadith, and Tasbeeh.

Flutter’s UI framework helps create a smooth and responsive interface.

---

# Application Workflow

1. The user launches the application.
2. The main screen displays the different Islamic sections available in the app.
3. The user selects a feature such as Quran, Hadith, or Tasbeeh.
4. The selected section opens and displays the relevant content.
5. The user interacts with the content by reading, browsing, or using the tasbeeh counter.

This workflow provides a simple and intuitive user experience.

# Project Structure

The project follows the standard Flutter project structure:

Contains the main source code of the application including screens, widgets, and business logic.

**assets/**
Contains images, fonts, and other static resources used in the application.

**android/**
Contains Android-specific configuration and platform files.

**ios/**
Contains iOS-specific configuration and platform files.

**test/**
Contains unit tests for the application.

This structured architecture helps developers maintain and extend the project more easily.

# How to Run the Project

To run the project locally on your machine, follow these steps:

### 1. Clone the repository

```
git clone https://github.com/mahmoud66844/Islami_App.git
```

### 2. Navigate to the project directory

```
cd Islami_App
```

### 3. Install dependencies

```
flutter pub get
```

### 4. Run the application

```
flutter run
```

Make sure that Flutter is installed and properly configured before running the project.
